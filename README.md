# 📱 Mobile Product Segmentation & Recommendation System

## 📌 Project Overview

This project focuses on analyzing global mobile product reviews using Python, Machine Learning, and Streamlit.

The dataset contains mobile phone reviews, ratings, pricing information, product specifications, and customer sentiment data collected from multiple brands and platforms.

The project performs data preprocessing, exploratory data analysis (EDA), product segmentation using K-Means Clustering, and builds a recommendation system using Cosine Similarity.

An interactive Streamlit dashboard is developed to visualize insights and recommend similar mobile products.

---

## 🚀 Tech Stack

- Python (Pandas, NumPy) – Data preprocessing & analysis
- Scikit-learn – Machine Learning & Recommendation System
- Matplotlib & Seaborn – Data visualization

## 📊 Cluster Insights

### Cluster 0 – Mid-Range Mobiles
Balanced smartphones with moderate pricing and average feature ratings.

### Cluster 1 – Premium High-Performance Mobiles
Highly rated smartphones with excellent performance, camera, battery, and display quality.

### Cluster 2 – Low Satisfaction Segment
Products with lower customer satisfaction and weak feature ratings.

### Cluster 3 – Ultra Premium Segment
Expensive smartphones with above-average performance and premium positioning.
